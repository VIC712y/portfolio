資料工程領域的實際專案經驗，包含 ETL 流程設計、資料視覺化

不動產實價登錄視覺化

1. 資料收集與爬蟲

使用 Python 開發爬蟲程式，從實價登錄系統獲取房地產交易資料。 實作座標轉換功能，將地址轉換為經緯度座標。 

def coordinates(add):
    url = f"https://api.nlsc.gov.tw/idc/TextQueryMap/{add}"
    headers = {
        "Content-Type": "application/xml",
        "Referer": "victor.proagent.com.tw",
    }
    try:
        response = requests.get(url, headers=headers)
        response.raise_for_status()
        root = ET.fromstring(response.text)
        location = root.find('.//LOCATION').text
        location = location.split(",")
        return location
    except requests.exceptions.RequestException as e:
        print("Error occurred:", e)

2. 資料清洗與前處理

對爬取的原始資料進行清洗和標準化，處理缺失值、異常值， 並進行必要的資料轉換和編碼。主要工作包括：

  • 移除重複和無效資料
  • 統一日期和金額格式
  • 處理特殊字元和編碼問題
  • 補充缺失的地理資訊

3. ETL 流程實作

設計並實現 ETL 流程，將處理後的資料存入 SQL 資料庫。 建立自動化流程，確保資料的即時性和一致性。

4. Tableau 視覺化設計

使用 Tableau 建立互動式儀表板，包含：

  • 房價趨勢分析
  • 地理位置分布圖
  • 房型和價格關係分析
  • 交易量變化趨勢

5. 系統維護與優化

負責維護和優化整個系統，包括資料更新機制、效能調校， 以及根據使用者回饋持續改進視覺化呈現方式。
